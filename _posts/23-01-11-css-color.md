# color

The 'color' property sets the element's text and text decoration color.
There are several ways to show the color.

# Syntax

```css
color: color;
```

# Example

```html
<ul>
  <li id="red">red</li>
  <li id="blue">blue</li>
  <li id="orange">orange</li>
  <li id="pink">pink</li>
</ul>
```

```css
#red {color: red;}
#blue {color: #0000FF;}
#orange { color: rgb(255,165,0);}
#pink {	color: rgb(100%, 75.3%, 79.6%); }
```
