# NOT

The NOT operator returns 0 when the bit is 1. 
And if the bit is 1, it returns 0.

# Syntax

```javascript
~a
```

|a|NOT a|
|-|-|
|0|1|
|1|0|

# Example 

```javascript
 9 (base 10) = 00000000000000000000000000001001 (base 2)
               --------------------------------
~9 (base 10) = 11111111111111111111111111110110 (base 2) = -10 (base 10)
```
