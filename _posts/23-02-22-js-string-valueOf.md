# String.prototype.valueOf&#40;&#41;

It returns the value of string object.


# Syntax

```javascript
valueOf()
```

# Example
Example is from mdn page of String.prototype.search.

```javascript
const str = new String("Hello world");
console.log(str.valueOf()); // 'Hello world'
```
