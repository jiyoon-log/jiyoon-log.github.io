In this post, it tells you about some of 'transform'.

# transform

It allows elements to be rotated, resized, tilted, and moved.


# Example

```html
<div class="container"> </div>
```

```css
.container {
  width:100px;
  height: 100px;
  background-color: purple;
  transform: translate(20px, 30px) rotate(30deg);
}
```
