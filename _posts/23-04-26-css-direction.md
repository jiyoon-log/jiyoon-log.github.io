In this post, it tells you about some of 'direction'.

# direction

It sets the direction of text, table columns.
(From reference page)


# Example

```html
<div class="ltr">  1 2 3 4 5 </div>
<div class="rtl">  1 2 3 4 5 </div>
```

```css
.ltr { direction: ltr;}
.rtl { direction: rtl;}
```
